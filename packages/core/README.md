# @flowness-labs/core

> **Flowness**: The Issue-driven AI Development Operating System.

This package contains the core logic, configurations, state managers, and templates for Flowness. It is used by the CLI and other system modules to initialize workspace scaffolds and enforce workflow structures.

---

## What is Flowness?

Flowness is an AI agent orchestration system that enforces software development workflows and guidelines on AI agents.

Every request becomes an **Issue**, and every step produces immutable **Evidence** under an append-only log model.
