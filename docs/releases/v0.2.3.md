# Flowness v0.2.3 Release Notes

- Date: `2026-06-21`
- Scope: 패치 릴리스 (업그레이드 시 누락된 rules/workflows 추가 버그 수정)

## 요약 (Summary)

기존 프로젝트 업그레이드 시 `compareOnlyArtifacts`(rules 및 workflows 파일들) 중 사용자 워크스페이스에 존재하지 않는 파일들이 `Will add if missing` 목록에 자동으로 추가되지 않아, 업그레이드 완료 후 `flowness validate` 실행 시 `.flowness/rules/git.md` 등의 파일 누락 오류가 발생하는 버그를 수정했습니다.

## Fixed

- `flowness upgrade` 시 워크스페이스에 존재하지 않는 필수 `compareOnlyArtifacts`(규칙 및 워크플로 템플릿) 파일들을 누락 파일 목록(`addIfMissing`)에 추가하여 안전하게 업그레이드되도록 버그를 수정했습니다.

## Migration Notes

- 기존 프로젝트에서 `flowness upgrade --apply`를 다시 수행하면, 누락되었던 `.flowness/rules/git.md` 등의 파일이 안전하게 추가되며 이후 `flowness validate` 통과가 가능해집니다.
