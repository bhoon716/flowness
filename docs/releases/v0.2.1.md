# Flowness v0.2.1 Release Notes

- Date: `2026-06-21`
- Scope: 핫픽스 릴리스 (전역 CLI 설치 실패 및 런타임 의존성 메타데이터 수정)

## 요약 (Summary)

`@flowness-labs/workflow-engine@0.2.0`에서 런타임 중에 `typescript` 패키지를 가져오지만, 패키지 메타데이터의 `dependencies`에 `typescript`가 누락되어 전역 설치 시 실행 오류가 발생하는 문제를 해결한 핫픽스 릴리스입니다.

## Migration Notes

- 기존 0.2.0 버전을 사용 중이던 환경에서는 글로벌 CLI를 0.2.1 버전으로 업데이트하여 설치 문제를 해결할 수 있습니다.
