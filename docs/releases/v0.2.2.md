# Flowness v0.2.2 Release Notes

- Date: `2026-06-21`
- Scope: 패치 릴리스 (버전 정합성 및 업그레이드 버그 수정, 버전 옵션 추가)

## 요약 (Summary)

기존 프로젝트 업그레이드 도중 발견된 기본 타겟 버전 오류를 수정하고, CLI에 버전 출력 명령(`--version`, `-v`)을 정식 추가하였으며, 업그레이드 시 누락된 파일 추가 리스트에서 발생하는 중복 문제를 해결했습니다.

## Added

- `flowness --version` 및 `flowness -v` 명령어를 추가했습니다. 컴팩트하게 버전 명칭만 출력합니다.

## Changed

- 업그레이드 계획 내 `Will add if missing` 목록에 중복 경로가 나오지 않도록 개선했습니다.

## Fixed

- `flowness upgrade` 기본 타겟 버전이 하드코딩된 `0.1.5`가 아닌 현재 CLI 버전을 정확히 반영하도록 버그를 수정했습니다.

## Migration Notes

- 기존에 초기화된 프로젝트는 `flowness upgrade --apply`를 호출하여 `.flowness/harness-manifest.json` 등의 파일을 0.2.2 버전 기준으로 최신화할 수 있습니다.
